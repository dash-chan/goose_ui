# goose_ui
an elegant uikit for flutter desktop. 

## Getting Started

### run gallery 

```shell
cd gallery
flutter run
```

* generate gallery code
> using codeviewer_cli
```dart
dart run tool/codeviewer_cli/main.dart
```

### development

```dart
melos bs
```
